# 🦐 小虾秘网页助手 - Chrome/Edge 浏览器插件

一键抓取网页正文内容，推送到飞书机器人的浏览器插件。

## ✨ 功能特性
- 🎯 **纯前端实现**：Vue3 + Element Plus CDN引入，无需打包工具
- 📝 **智能内容提取**：集成Mozilla Readability库，自动提取网页正文
- 🔒 **本地配置存储**：飞书机器人配置保存在浏览器本地，安全可靠
- 🎨 **卡通风格UI**：渐变紫色主题，友好的交互体验
- 📤 **一键推送**：点击按钮即可将当前页面内容推送到飞书群
- 🔔 **实时反馈**：推送成功/失败都有清晰的提示信息

## 🚀 安装方法
1. 打开Chrome/Edge浏览器，进入扩展管理页面（地址栏输入 `chrome://extensions/` 或 `edge://extensions/`）
2. 开启右上角的「开发者模式」
3. 点击左上角「加载已解压的扩展程序」，选择当前 `browser-extension` 文件夹
4. 插件就会出现在浏览器扩展栏中啦

## ⚙️ 配置说明
1. 点击插件图标打开侧边栏
2. 在「飞书机器人配置」中填写：
   - **Webhook**：飞书自定义机器人的Webhook地址
   - **加签密钥**：飞书机器人的加签密钥（如果开启了签名验证的话）
3. 点击「保存配置」按钮保存设置

## 📖 使用方法
1. 打开任意你想要推送的网页
2. 点击插件图标，点击「📤 推送当前页面到飞书」按钮
3. 等待几秒，内容就会自动推送到你的飞书群里啦

## 📦 技术栈
- **Manifest V3**：最新的Chrome扩展标准
- **Vue 3**：渐进式JavaScript框架
- **Element Plus**：Vue 3组件库
- **Readability**：Mozilla开源的网页正文提取库
- **CryptoJS**：加密库，用于飞书机器人签名验证

## 📝 注意事项
- 飞书机器人需要先在飞书群中创建，并且开启「自定义机器人」功能
- 如果开启了飞书机器人的IP白名单，请确保浏览器所在网络在白名单内
- 推送的内容会自动截取前2000字符，超过部分会省略
- 插件需要获取当前网页的读取权限，才能提取正文内容
